Connect to the cloud linux desktop

1. Download and install TigerVNC. Click the following link to download the appropriate installer depending on your OS:

  • MacOS (intel)
  • Windows
Apple silicon user (M1/M2) users: TigerVNC has been developed for older models using intel chips. You can still use it, but you will need to install Rosetta 2 first. Alternatively, you can use the built in VNC client on MacOS. Just open a browser and type in the following address vnc:// followed by the IP address of the cloud machine in thee address bar. This should open a VNC client for you.

2. Start TigerVNC and enter the IP address that you have been provided and click connect.



3. Enter the password: genomics


4. You should now see your virtual desktop and can start using it.
